    Linux Installation Steps (CLI):
    Open your terminal and run these commands (requires sudo privileges for global install): 
    curl -fL "https://github.com/dipesh-m12/Go-backend-and-tool/releases/download/linux_amd64/panty" -o /tmp/panty_tool_linux
sudo mv /tmp/panty_tool_linux /usr/local/bin/panty

panty --version

    This will download, move, and make `panty` executable, allowing you to run it from any directory.

    `panty` is a powerful command-line interface (CLI) tool designed for load and health testing of local servers. It supports concurrent requests, customizable HTTP methods and data, and real-time health monitoring with automatic termination on failure.
