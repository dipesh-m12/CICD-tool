  • CDN Server
    View on GitHub

    A custom CDN server designed to provide free CDN tiers for services like Firebase and others that don't offer them natively. It drastically decreases content loading time from 13 seconds to 20 milliseconds. Key features and usage: 

    Features: In-memory Caching | Configurable Cache Duration | Configurable Max Cache Size | Automatic Content-Type Detection | Local File Serving | External Content Proxying

Usage:
  1. Navigate to the project directory.
  2. Install dependencies: npm install (assuming Node.js project)
  3. Start the server: node server.js (or npm start if configured)

Commands (after server is running):
  - Access local files: http://localhost:3000/filename.ext
  - Access external content: http://localhost:3000/proxy?url=https://example.com/image.jpg
