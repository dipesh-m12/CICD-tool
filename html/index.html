    Windows Installation Steps:
    Follow these steps to install and configure the CICD Tool on Windows: 
    1. Download the CICD Tool installer from the link above.
2. Run the installer (mysetup.exe) and follow the prompts to install to C:\Program Files\GoTools.
3. Add the tool to your system PATH:
   - Right-click the Start button and select "System".
   - Click "Advanced system settings" > "Environment Variables".
   - Under "System variables", select "Path" and click "Edit...".
   - Click "New" and add: C:\Program Files\GoTools
   - Click "OK" to save.
4. Open a Command Prompt and run:
   cicd launch -p 8080
    Linux Installation Steps (CLI):
    Open your terminal and run these commands (requires sudo privileges): 
    curl -fL "https://github.com/dipesh-m12/CICD-tool/releases/download/cicd-linux-amd64/cicd" -o /tmp/cicd_tool_linux
sudo mv /tmp/cicd_tool_linux /usr/local/bin/cicd
sudo chmod +x /usr/local/bin/cicd
cicd launch -p 8080
